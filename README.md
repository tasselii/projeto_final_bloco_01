## 📌 Descrição

O **Projeto Sorveteria Admin** é um sistema de gerenciamento de sorveteria desenvolvido em **Java**, com ênfase total na **Programação Orientada a Objetos (POO)**. O projeto simula o funcionamento interno de uma sorveteria, permitindo:

- 🧑‍💼 Acesso como **Administrador** para gerenciar produtos e pedidos;
- 🛍️ Acesso como **Cliente** para realizar compras e visualizar cardápio;
- 🔐 Um menu principal intuitivo e executado no console, sem uso de métodos separados para menus (estrutura única de execução).

Foi uma oportunidade prática de aplicar diversos conhecimentos fundamentais de Java e estrutura de sistemas.

---

## 🚀 Funcionalidades

### Modo Cliente
- 📜 Listar produtos
- ➕ Adicionar itens ao pedido
- 💵 Finalizar compra
- 🛒 Visualizar e gerenciar o carrinho de compras

### Modo Admin
- ➕ Cadastrar produtos
- 🔍 Consultar produto por ID
- 📋 Listar todos os produtos
- ✏️ Atualizar produto
- ❌ Deletar produto
- 📊 Ver compras realizadas

---

## 🧩 Tecnologias e Conceitos Aplicados

O projeto foi elaborado aplicando os principais fundamentos da linguagem Java:

- 🧠 Programação Orientada a Objetos (POO)  
- 🔄 Estruturas de decisão e repetição  
- 🧮 Manipulação de arrays e listas (`ArrayList`)  
- 💥 Tratamento de exceções (`try/catch`)  
- 📦 Organização por pacotes e boas práticas de modularização  
- 💬 Interface de usuário via console com menus interativos

Desenvolver esse projeto foi essencial para fortalecer minhas habilidades com:

---

## 🧗 Desafios Enfrentados

- 🔁 Criar menus e navegação em um **único método principal**, mantendo a legibilidade  
- 🧮 Lidar com **operações em lista de objetos**, como cadastro e exclusão de produtos  
- 🧪 Garantir a integridade dos dados ao simular um processo de vendas  
- 🔐 Manter o sistema funcional mesmo com entradas inválidas, utilizando `Scanner` e tratamento de exceções  

Esses obstáculos foram essenciais para me preparar para desafios reais de desenvolvimento de sistemas.
